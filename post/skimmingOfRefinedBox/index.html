Skimming - RefinedBox: Refining for fewer and high-quality object proposals

2020-07-03 7 min read # Notes

论文解决的任务

自底向上的似物性采样(object proposals)方法存在的缺陷

Since it is difficult to represent high-level semantic information using traditional hand-crafted features, these bottom-up methods usually i) fail to rank the generated proposals properly and ii) have to use large number of proposals to ensure detection recall.

  1. 无法对产生的候选框进行正确排序
  2. 须使用数量庞大的候选才能保证一定的检测召回率（精度不高）

基于深度学习的proposal method存在的缺陷

Hence the sub-optimal proposal sampling strategies make them difficult to fully leverage the powerful capability of CNNs. As a result, the number of true objects (e.g. usually less than 10) in an image is still much smaller than the number of proposals generated by these deep-based methods (e.g. usually a few hundred).

由于次优的候选采样策略无法充分利用卷积神经网络的强大力量，生成的候选数量远大于实际对象的数量。

解决方法

To combine the superiority of traditional proposal methods and the powerful representation capability of CNNs, we propose a novel method to re-rank and align existing proposal boxes in a single inference of a neural network.

The proposed network can share convolutional features with other high-level tasks by joint training, so the proposal refinement can be very fast.

  1. 目标

    在保持高检测召回率的前提下减少候选数量

  2. 途径

    设计了一个计算轻量级的神经网络来精炼最初的目标候选集

    • 候选重排 - 根据覆盖完整对象的紧密程度对候选进行排序

    • 边框回归 - 微调候选位置与形状，使其能更紧密地覆盖真实物体

    该神经网络能够通过联合训练与其他高层任务共享卷积特征，以提高候选的优化速度。

使用的模型

网络结构

we build our network based on VGG16 to showcase our refinement method. Our network takes a natural image and corresponding initial boxes as input.

基于VGG16 (含13个卷积层和3个全连接层) 构建神经网络，以自然图像和由其他对象候选框生成方法产生的initial boxes作为输入。

Thus, we first connect a convolutional layer with kernel size 3 × 3 after the 13-th convolutional layer to reduce the number of channels from 512 to 128. Then, a ROI Pooling layer is followed to down-sample each initial box region into a fixed feature map size, i.e. 7 × 7. ROI Pooling divides an input feature map into grids with the same width and height and perform max pooling in each grid. Next, a fully connected layer with only 512 output neurons is connected. A ReLU layer is followed after the added convolutional layer and fully connected layer, respectively. At last, two branches of ranking and box regression are used to recompute the objectness score and obtain the location offsets of each initial box.

RefinedBox的结构

  1. 在VGG16的第13层卷积层后连接一个卷积核大小为3 * 3的卷积层，将通道数由512降至128 (后随一ReLU层(线性整流层))
  2. 后接ROI Pooling层，输出大小固定的特征映射
  3. 连接带有512个输出神经元的全连接层 (后随一ReLU层)
  4. 使用ranking和box regression两个分支来进行对象得分 (objectness score) 的重新计算以及对任意initial boxes位置偏移量的获取

The branch of RefinedBox is designed to refine the initial boxes, then the refined boxes are inputted into the branch of Fast R-CNN for classification.

we connect the well-known detection framework, Fast R-CNN, after the convolutional layers as a parallel branch to RefinedBox. The refined proposals produced by the RefinedBox branch are inputted into Fast R-CNN.

将Fast R-CNN作为基础卷积层后与RefinedBox相平行的分支，经过RefinedBox精炼后的候选们作为Fast R-CNN的输入。

使用的数据集

We evaluate the proposed method on two widely used object detection datasets, including PASCAL VOC2007 and MS COCO.

  1. PASCAL VOC2007
    • 由训练集（2501幅训练图像及2510幅验证图像）和测试集（4952幅测试图像）组成，共包含20个种类
    • 在VOC2007训练验证集上训练RefinedBox，在VOC2007测试集上进行测试
  2. MS COCO
    • 包含82,783幅训练图像和40,504幅验证图像
    • 采用其训练集训练RefinedBox，采用其验证集进行对候选(proposals)进行评估

度量

To evaluate the proposals, we adopt the metrics of object detection recall (DR), mean average best overlap (MABO), and average recall (AR).

DR (对象检测召回率), MABO (所有类别的平均最高重叠率), AR(平均召回率)

结论

  1. RefinedBox可以有效地精炼给定原始候选集，且因为用于精炼的网络易于被优化，使其可以与后续的应用进行联合训练

  2. 局限性

    1. 因RefinedBox的效率与初始的候选数目呈负相关，当初始的候选数目过多时存在效率低下的问题
    2. 在作用于存在过多小物体的图像时RefinedBox的性能会受到影响

  3. 未来的方向

    In the future, we plan to apply our refinement method to other high-level applications, e.g. mining knowledge from huge amounts of unlabeled data.

    在更多的高级应用上使用RefinedBox。
